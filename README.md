# ⚛ Gemini React Agent

Generate production-ready React JSX pages using Google Gemini AI.

## 🚀 Deploy to GitHub Pages

### Step 1 — Create repo
1. Go to https://github.com/new
2. Name it exactly: `gemini-react-agent`
3. Set to **Public** → Create

### Step 2 — Upload files
Upload all files keeping folder structure (use GitHub Desktop for easiest upload)

### Step 3 — Enable GitHub Pages
Settings → Pages → Source → **GitHub Actions** → Save

### Step 4 — Done!
Your site: `https://YOUR-USERNAME.github.io/gemini-react-agent/`

## 🔑 Get your Gemini API Key
1. Go to https://aistudio.google.com/app/apikey
2. Click **Create API Key**
3. Copy and paste into the app — stored only in your browser

## 💻 Run locally
```bash
npm install
npm run dev
```
